# 💼 Chamsseddine Douiri - Portfolio

A personal web portfolio designed to showcase my projects, skills, and experience as a full-stack web developer.  
Built with **Tailwind CSS**, it features a responsive layout, a clean UI, and multi-language support (EN/FR).

## 🚀 Technologies Used

- **HTML5 & Semantic Markup**  
- **Tailwind CSS**  
- **JavaScript (Vanilla)**  
- **Font Awesome**  
- **Responsive Design**  
- **Git & GitHub**
